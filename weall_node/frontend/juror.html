Load dispute case

Paste a case id from the disputes or recovery system. For recovery cases, the case_type will be "recovery".

Decision

Only open cases can be decided. This console will call /disputes/cases/{case_id}/decide, which triggers the correct enforcement (for example, recovery key rebinding for recovery cases).

Decisions are final at this layer. Additional appeal or override flows should be encoded in higher-level governance processes, not here.
